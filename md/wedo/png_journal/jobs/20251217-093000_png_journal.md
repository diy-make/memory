# PNG Journal WeDo Job

## Metadata
- **Description:** Scans the root PNG inbox, processes images into a daily journal, and archives them in a chrono-fractal structure.
- **Version:** 1.1
- **Preferred_Agent:** gemini-2.5-flash
- **Reason:** High speed and low cost are sufficient for the image description and summarization required in this batch processing task.

- [x] [IMG-01] **STEP 1:** EXECUTE `list_directory` dir_path=repos/diy-make/memory/public/png output=image_list

- [ ] [IMG-02] **STEP 2:** LOOP image_list as image (Auto-Approve Mode):
    - [ ] [IMG-02a] **STEP 2a:** IF image ends with ".png" AND image != "01-a-screenshot-of-a-computer-screen-likely-showing-c.png":
        - [ ] [IMG-02b] **STEP 2b:** EXECUTE `read_file` file_path="repos/diy-make/memory/public/png/{image}" output=desc_raw agent="gemini-2.5-flash" reason="High speed/low cost sufficient for OCR/description"
